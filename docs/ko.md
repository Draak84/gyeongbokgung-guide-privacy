# 개인정보 처리방침 — Gyeongbokgung Guide

*최종 업데이트: 2026-04-26*

본 정책은 모바일 애플리케이션 **Gyeongbokgung Guide**(이하 "본 앱")의 개인정보 수집·이용에 관한 사항을 안내합니다. 본 앱은 외국인 관광객 및 일반 사용자에게 경복궁 탐방 정보를 제공하기 위해 만들어졌으며, 사용자의 콘텐츠 이용 데이터를 자체 서버에 저장하지 않습니다.

## 1. 수집하는 정보

| 항목 | 수집 시점 | 용도 | 사용자 동의 |
|---|---|---|---|
| **광고 식별자 (IDFA)** | 앱 첫 실행 시 ATT 동의를 받은 경우에만 | Google AdMob 개인화 광고 송출 | 명시적 동의 (거부 시 비개인화 광고로 동작) |
| **대략적 위치 정보** | 사용자가 위치 권한 허용 시에만 | 지도 화면에 현재 위치 표시 | 명시적 동의 (거부 시 지도 사용 가능, 위치 표시만 비활성) |
| **진단 데이터** (크래시 로그, 디바이스 모델, OS 버전) | 앱 비정상 종료 시 자동 | 앱 안정성 개선 (Firebase Crashlytics) | 자동 (개인 식별 정보 미수집) |
| **앱 사용 분석** (화면 진입, 버튼 클릭 등) | 사용 중 자동 | 기능 개선 (Firebase Analytics) | 자동 (개인 식별 정보 미수집) |

본 앱은 사용자의 이름·이메일·전화번호·연락처·사진·연락기록 등 **개인 식별 정보를 직접 수집하거나 자체 서버에 저장하지 않습니다**.

## 2. 사용자 권리

- **ATT(앱 추적 투명성) 거부**: iOS 설정 → 개인정보 보호 및 보안 → 추적 → 본 앱 비활성화. 거부 시 광고는 계속 표시되나 비개인화 광고로 송출됨.
- **위치 권한 거부**: iOS 설정 → 개인정보 보호 및 보안 → 위치 서비스 → 본 앱 → "허용 안 함". 거부 시 지도는 정상 동작하되 현재 위치 표시 기능만 비활성화됨.
- **알림 권한**: 본 앱은 푸시 알림을 사용하지 않습니다.

## 3. 데이터 위탁(제3자 제공)

본 앱은 다음 외부 서비스를 사용하며, 해당 서비스의 개인정보 처리방침을 따릅니다.

| 서비스 | 제공자 | 용도 |
|---|---|---|
| **Google AdMob** | Google LLC | 광고 송출 (IDFA 사용) — [정책](https://policies.google.com/privacy) |
| **Google Maps SDK** | Google LLC | 지도 타일 표시 — [정책](https://policies.google.com/privacy) |
| **Firebase Crashlytics** | Google LLC | 크래시 리포트 (IDFA 미사용) — [정책](https://firebase.google.com/support/privacy) |
| **Firebase Analytics** | Google LLC | 사용 패턴 분석 (IDFA 미사용) — [정책](https://firebase.google.com/support/privacy) |

본 앱이 직접 사용자 데이터를 외부 도메인으로 송신하는 경로는 위 4개 서비스에 한정됩니다.

## 4. 데이터 보관

- **앱 내 저장**: 사용자가 선택한 언어, 방문 명소 기록, 광고/위치 권한 상태 등은 사용자 기기 내부(iOS UserDefaults)에만 저장되며 외부로 송신되지 않습니다.
- **앱 삭제 시**: 위 모든 기기 내 데이터는 즉시 삭제됩니다.
- **자체 서버 미운영**: 본 앱은 자체 백엔드 서버를 운영하지 않으며, 모든 콘텐츠(명소 정보, 이미지)는 앱 번들 안에 포함되어 있습니다.
